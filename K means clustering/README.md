This project demonstrates **K-Means clustering** using **scikit-learn** on a synthetic dataset generated with `make_blobs`.  
The clustered data points and their centroids are visualized using Matplotlib.

---

## Dataset

- **Source:** `sklearn.datasets.make_blobs`
- **Samples:** 300
- **Clusters:** 3
- **Cluster Standard Deviation:** 1.0
- **Random State:** 42

---

## Model Details

- **Algorithm:** K-Means Clustering
- **Number of Clusters (`k`):** 3
- **Initialization Runs (`n_init`):** 10
- **Random State:** 42

---

- Python 3.7+
- NumPy
- scikit-learn
- matplotlib
